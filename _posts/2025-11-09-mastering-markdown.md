Markdown is a lightweight markup language with plain-text-formatting syntax. It's the format I use for writing all the posts on this blog, and it's one of the most popular markup languages in the world. Its simplicity and readability make it a joy to work with.

### Why Markdown?

*   **Simple and Clean:** The syntax is intuitive and doesn't get in the way of the content.
*   **Platform Independent:** Markdown can be converted to many other formats, including HTML.
*   **Readable:** The raw Markdown text is almost as readable as the final rendered output.

### Basic Syntax

Here are a few examples of common Markdown syntax.

#### Headers

```markdown
# This is an H1
## This is an H2
### This is an H3
```

#### Emphasis

You can make text *italic* or **bold**.

```markdown
*This text will be italic*
_This will also be italic_

**This text will be bold**
__This will also be bold__
```

#### Lists

You can create ordered and unordered lists.

```markdown
1. First item
2. Second item
3. Third item

- Unordered item
- Another item
```

#### Links and Images

```markdown
[Jekyll TeXt theme](https://kitian616.github.io/jekyll-TeXt-theme/)

![A placeholder image](https://via.placeholder.com/150)
```

This is just a small sample of what Markdown can do. To learn more, check out the [Markdown Guide](https://www.markdownguide.org/).
